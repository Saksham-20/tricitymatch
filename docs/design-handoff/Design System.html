Design System · v1

The TricityShadi system.

A premium, trustworthy, family-safe language for Tricity's matrimony platform. Burgundy is an accent, never a flat fill; gold marks premium. Playfair for headings, Inter for everything else. Every token below maps 1:1 to tailwind.config.js and index.css.

Usage rule. Large surfaces stay neutral-50 / white (dark: ink). Burgundy appears as accents, thin rails, tonal 600→800 gradients on small elements. Gold is reserved for premium / VIP. No pastel rainbow panels — one muted neutral info panel standard.
