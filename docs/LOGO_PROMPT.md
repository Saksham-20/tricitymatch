# TricityShadi Logo-Generation Prompt (Copy/Paste)

Use this prompt with any image-generation AI (Midjourney, DALL·E, Ideogram, Leonardo, etc.) to generate high-quality logo ideas.

```text
You are a senior brand identity designer. Create logo concepts for a website named “TricityShadi”.

ABOUT THE WEBSITE:
- TricityShadi is a modern matrimonial platform focused on the Tricity region of India: Chandigarh, Mohali, and Panchkula.
- Brand promise: trusted matchmaking with modern technology.
- The platform combines emotional connection, family values, and smart compatibility.
- Core features include AI-based compatibility matching, profile discovery, real-time chat for matched users, identity verification, and privacy controls.
- Audience: marriage-focused young professionals and families in North India, primarily users who value trust, seriousness, safety, and cultural relevance.

BRAND PERSONALITY:
- Trustworthy, warm, premium, modern, respectful, secure, and hopeful.
- Should feel culturally rooted but not old-fashioned.
- Avoid playful dating-app vibes; this is matrimonial, not casual dating.

LOGO GOAL:
Generate 8 distinct logo directions with visual rationale.
For each direction include:
1) Concept name
2) Symbol idea
3) Why it fits the brand
4) Suggested color palette (hex)
5) Recommended typography style
6) Monochrome usability notes
7) App icon adaptation idea

VISUAL THEMES TO EXPLORE:
- Subtle union/connection motifs (interlocking forms, linked paths, two elements becoming one)
- Abstract map/reference to the “tri-city” idea (three points/nodes/arches) without being literal or cartoony
- Premium trust cues (shield-like geometry, elegant symmetry, verification-inspired details)
- Modern tech + human warmth balance

STYLE REQUIREMENTS:
- Clean, minimal, vector-friendly, scalable from favicon to billboard
- Flat design preferred (no heavy gradients or 3D effects)
- Strong silhouette and recognizability
- Should look good in color, black, and white
- Avoid generic hearts/rings unless abstracted in a unique, modern way

COLOR DIRECTION:
Provide at least 3 palette families across the concepts:
- Trust + premium: deep navy, royal blue, muted gold
- Warm + human: maroon/rose + neutral cream/charcoal
- Modern + fresh: teal + indigo + soft gray
Keep contrast and accessibility in mind.

TYPOGRAPHY DIRECTION:
- Prioritize elegant sans-serif or humanist serif pairings
- Wordmark should be readable and premium
- Consider a custom “T” and “M” ligature option for one concept

OUTPUT FORMAT:
- Present 8 concepts in a structured table.
- Then provide a “Top 3 Recommended Concepts” section with pros/cons.
- Then provide “Refinement Suggestions” for the best concept:
  - Simplification ideas
  - Spacing/alignment tweaks
  - Alternate lockups (horizontal, stacked, icon-only)
  - Dark mode adaptation

FINAL DELIVERABLE:
At the end, write one final production-ready prompt I can use in an image generator to create the single best logo concept in multiple variations (white background, transparent background, monochrome, and app icon).
```
