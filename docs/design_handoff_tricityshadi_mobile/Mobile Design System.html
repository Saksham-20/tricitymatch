The web brand, translated into native idioms — bottom tabs, sheets, haptics, platform back. Burgundy stays an accent, gold means premium, Playfair carries the premium voice. Every token here maps 1:1 to the web system so both platforms read as one product.

01

Color & usage rules

Burgundy — primary accent

The brand voice. Used for borders, ticks, active states, links and the primary CTA. Never flat-filled across large areas — it's a signal, not a wallpaper.

Gold — premium signal only

Reserved exclusively for VIP, locks, "Most Popular" and boost. If it isn't about money or status, it isn't gold.

Neutrals & semantic

Warm-grey surfaces carry the UI. One muted info style — no rainbow of colored callouts.

Translucent, hairline border, back chevron left + 17pt title. Large-title style for tab roots; compact on push.

06

Motion & haptic tokens

Reanimated-buildable. Three durations, three curves, one spring, a fixed haptic map. Every interaction in the redlines references these names.

TokenValueUsed for
dur.fast 120msease-stdPress scale, tab tint, chip toggle
dur.base 240msease-stdSheet fade, screen content, toast
dur.slow 360msease-outRing fill, guna bars, skeleton → content
spring.popstiffness 380 · damping 18Like/shortlist pop, match seal
spring.sheetstiffness 240 · damping 28Bottom-sheet open/snap
haptic.lightselectionAsyncTab switch, chip, segmented
haptic.successnotification(Success)Interest sent, mutual match, payment
haptic.warningnotification(Warning)Form error shake, 429 lockout
haptic.mediumimpact(Medium)Long-press reveal, incoming call

Handoff. These tokens live in ts-mobile.css and map to the Expo theme. Every screen canvas (see all →) carries redline cards with token names, spacing in pt, and the motion/haptic per interaction — buildable directly in Reanimated + react-navigation.
